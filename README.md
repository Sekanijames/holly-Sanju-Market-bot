# Telegram Virtual Number Bot

A complete Telegram bot for buying virtual numbers with social media boosting, referral system, and admin panel.

## Features

✅ **Virtual Numbers**
- Integrate with BitRaHQ API
- Browse available services
- Purchase numbers with balance

✅ **Admin Panel**
- Set channel link for verification
- Update API keys
- Manage payment methods
- View dashboard statistics
- Monitor orders

✅ **Channel Verification**
- Users must follow channel before access
- Automatic verification check
- One-click join button

✅ **Referral System**
- Generate unique referral links
- Users refer 10 people to get reward
- 10% commission on first payments
- Use commission to buy numbers

✅ **Payment Integration**
- Support for multiple payment methods
- Order tracking
- Balance management
